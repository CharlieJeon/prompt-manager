# NAVER 파트너스룸 수업 예약 템플릿

## 기본 규칙

- 예약 가능 기간: 7일 이내
- 한 번에 1시간씩, 주 3회까지
- **예약 내용 입력 시에는 자동완성 목록에서 기존 예약 제목을 선택한다.**
  - “예약내용” 입력란에 첫 글자를 입력하면 최근 예약 내역이 나타납니다.
  - 동일한 수업명(예: ‘영어수업’, ‘일본어수업’)을 선택하여 입력을 확정합니다.
  - 이렇게 하면 시스템에서 필수 입력이 인식되고, 직접 타이핑할 때 발생할 수 있는 오류를 방지할 수 있습니다.

## 주간 수업 스케줄

| 요일   | 시간        | 수업명     |
| ------ | ----------- | ---------- |
| 월요일 | 12:50–13:50 | 영어수업   |
| 수요일 | 12:50–13:50 | 일본어수업 |
| 금요일 | 12:50–13:50 | 영어수업   |

## 장소 우선순위

1. 그린팩토리 파트너스룸 중 가장 작은 방
2. 그린팩토리에 빈 방이 없으면 1784 파트너스룸
